## Quick Start

1. Clone the repository
   
	```bash
	git clone https://github.com/your-username/myshelf.git
	cd myshelf
	```

2. Install dependencies

	frontend
	```bash
	cd frontend
	npm install
 	npm run dev
	```

3. Install dependencies

	backend
	```bash
	cd backend
	npm install
 	npm start
	```
